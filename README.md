# Dante's Star
A maze generator and solver in text format

## Usage
```bash
./generator x y [perfect]
```

x : lenght of the maze (x > 0),

y : width of the maze (y > 0),

perfect (optional) : create a perfect or imperfect maze.

```bash
./solver maze.txt
```

maze.txt : the maze in text format
